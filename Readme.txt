# Scripts
npm install -- to install all the dependencies

npm start -- to start the application

npm run ios -- to run the application in an iOS simulator or device

npm run android -- to run the application in an android simulator or device

npm test -- to run all the test suites


# Folder Structure
The root file is the App.tsx.The Screens folder include two screens- Home Screen and Trade History screen. The Containers folder includes UI components and API logic.
